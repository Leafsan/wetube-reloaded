# Wetube Reloaded

Cloning YouTube with Vanilla and NodeJS

/ -> Home  
/join -> Join  
/login -> Login  
/search -> Search  

/users/:id -> See User  
/users/logout -> Logout  
/users/edit -> Edit user  
/users/remove -> Delete user  

/videos/:id -> Watch Video  
/videos/:id/edit -> Edit Video  
/videos/:id/delete -> Delete Video  
/videos/upload -> Upload Video
